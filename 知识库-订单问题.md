# 订单问题知识库

## 一、订单状态说明

### 1.1 订单状态流转

```
待付款 → 待发货 → 已发货(物流中) → 已签收 → 交易完成
                                        ↓
                                   申请售后 → 退款/换货中 → 售后完成
```

### 1.2 常见订单状态

| 状态 | 说明 | 可执行操作 |
|-----|------|----------|
| 待付款 | 订单已创建，等待付款 | 付款、修改信息、取消订单 |
| 待发货 | 已付款，仓库准备发货 | 修改地址/电话、申请退款 |
| 已发货 | 仓库已出库，物流运输中 | 查询物流、修改地址（需拦截）、申请退款 |
| 已签收 | 快递已签收 | 确认收货、申请售后 |
| 交易完成 | 已确认收货 | 申请售后（15天内）、评价 |

## 三、各类订单问题处理

### 3.1 修改信息

**典型案例：修改收货地址**

用户："地址写错了，发到老地址去了，能改一下吗？"

**标准话术（待发货状态）：**
好的，您的订单目前是"待发货"状态，可以直接修改地址。请提供完整的新收货地址：
- 收货人：
- 联系电话：
- 详细地址：（省市区+街道门牌号）

我立即为您更新，不影响发货时间。

**标准话术（已发货状态）：**
您的订单已经发货了，包裹目前在物流途中。我需要联系中通快递进行拦截和转寄：
- 拦截成功率：约85%（越早申请越好）
- 可能产生的转寄费：5-10元
- 处理时效：提交后1-2小时内反馈结果

请提供新地址，我现在就提交拦截申请。如果拦截失败，我们可以协商退货重发。

### 3.2 催促发货

**典型案例：**

用户："都过去三天了还没发货，到底什么时候发？"

**标准话术：**
理解您等待的心情！查看到您的订单下单时间是4月10日，我们承诺48小时内发货（工作日），目前确实接近时限了。

我已为您在系统内加急标注（加急单号：JJ20260413-001），同时联系了仓库优先打包。预计今天下午15:00前出库，物流取件后会自动推送物流单号给您。

如果今天18:00前仍未收到物流信息，请再联系我，我将直接协调仓库主管处理。

**常见发货延迟原因及话术：**

| 原因 | 话术 |
|-----|------|
| 库存不足，等补货 | "该商品销量火爆，仓库正在紧急补货，预计X天后发出。如您着急，可选择换其他在库同类商品或申请退款。" |
| 定制商品生产中 | "您的定制订单正在生产中，当前进度约XX%，预计X月X日完成。完成后立即发出。" |
| 大促期间爆仓 | "由于近期订单量激增，发货有所延迟，已安排加班加点处理。您的订单已优先标记，预计明天发出。" |
| 节假日 | "因节假日仓库休息，您的订单将在X月X日（工作日第一天）优先发出。" |

### 3.3 退款/退货

**典型案例：未发货取消订单**

用户："不想要了，还没发货吧，直接退款"

**标准话术：**
好的，您的订单目前是"待发货"状态，可以直接取消退款。

退款将原路返回到您的支付账户：
- 支付宝/微信支付：预计1-3个工作日到账
- 银行卡：预计3-5个工作日到账

我已为您提交取消申请，退款到账后会有短信通知。如有其他需要随时联系我。

**典型案例：已收货退货**

用户："东西不太满意，想退货"

**标准话术：**
好的，我们支持7天无理由退换货（商品未使用、包装完好）。

**退货流程：**
1. 请在订单详情中点击"申请退货退款"
2. 选择退货原因并提交
3. 商家审核通过后（通常2小时内），会提供退货地址
4. 请将商品连同包装完整寄回
5. 我们收到退货并验收后，3个工作日内原路退款

**退货地址**：浙江省义乌市北苑街道春晗路168号 果冻鼠玩具厂仓库 陈先生收 17855812714

退回运费需要您先垫付（约8-12元），如果是质量问题退货，我们会补偿运费。

### 3.4 商品/物流异常

**标准话术：**
非常抱歉！请您在1小时内拍照上传以下凭证：

**商品问题拍照要求：**
1. 商品整体全景照（含包装）
2. 问题部位特写（破损/缺陷/色差等）
3. 外包装照片
4. 快递面单照片（含物流单号）

**物流异常截图要求：**
1. 物流详情页面截图（显示长时间未更新）
2. 订单详情截图

凭证上传后，我将在2小时内给出处理方案（换货/补发/退款三选一）。紧急问题可拨打17855812714直接联系商务经理。

### 3.5 发票问题

**标准话术：**
好的，为您处理发票。我们支持以下发票类型：

| 发票类型 | 所需信息 | 开票时效 |
|---------|---------|---------|
| 增值税普通发票（电子） | 公司名称 + 税号 + 邮箱 | 付款后7个工作日 |
| 增值税专用发票 | 公司名称 + 税号 + 地址电话 + 开户行及账号 + 邮寄地址 | 付款后10个工作日 |

请提供您需要的发票类型和开票信息，电子发票将发送至您的邮箱，纸质专票将邮寄到您指定地址。

**常见问题：**

**Q：可以补开之前订单的发票吗？**
A：可以，3个月内的订单均可补开。请提供订单号和开票信息。

**Q：发票抬头写错了能改吗？**
A：已开出的发票需要作废重开，请提供正确信息，3个工作日内重新开具。
