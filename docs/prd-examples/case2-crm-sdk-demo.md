# PRD 风格参考案例 — CRM SDK Integration Demo

> 此文件作为 PM Strategist Agent 的 PRD Style Learning 案例。

## 案例来源
- 产品线：TCCC Demo（Voice AI / Desk AI / Human Agent）
- 需求类型：Demo 页面新增（CRM SDK 集成展示）
- 特征：完整页面结构、SDK 功能模拟、引导教程系统、大量截图+文案对照表

## 提取的风格特征（Style Profile）

```
文档结构：需求所属客户 → 需求背景 → 需求描述 → 页面结构概览(层级展开) → 文案表(按模块分表)
语气：产品交付文档风格，有明确的用户路径和转化路径描述
术语：SDK、MCB(Mini Call Bar)、ACW(话后处理)、DTMF、CRM弹屏、技能组
详细程度：每个功能有独立表格行（功能名称 | 功能描述/交互 | 截图），极其详细
特殊格式：
  - 页面结构分层：主页面 → CRM Demo → 通话功能 → 座席面板 → 引导教程
  - 每个 SDK 功能单独一行描述
  - 文案按模块分组（座席面板/拨号盘/通话状态/MCB/转接/咨询/自助服务/DTMF...）
  - 中英文对照表按功能区域独立成表
  - 引导教程逐步骤描述（步骤号+功能名+交互+容错）
  - 用户体验路径和转化路径显式定义
```

---

## 原始需求单内容（摘要）

### 一、需求所属客户
对内架构师，国际站海外用户

### 二、需求背景
在现有 TCCC Demo 基础上新增 Human Agent 板块，通过完整的 CRM SDK 集成 Demo 展示 TCCC SDK 嵌入第三方业务系统的能力。

### 三、需求描述

**用户体验路径**：首页导航 → 点击"HUMAN AGENTS" → CRM SDK Demo → 引导教程 → 自由探索 → 跳转控制台

**用户转化路径**：CRM Demo → 了解 SDK 能力 → 跳转控制台 → 应用开通 → 免费试用

### 四、页面结构概览

#### 4.1 主页面新增
侧边栏导航新增"03 HUMAN AGENTS"，与 Voice/Desk 同级

#### 4.2 CRM Demo 应用

**4.2.1 整体布局**：顶部导航栏 + 浮动按钮
**4.2.2 CRM 页面功能**：客户详情(含Event Log) + 联系人表格
**4.2.3 SDK 通话功能（12项）**：
- 模拟呼入来电、接听+CRM弹屏、一键外呼
- MCB通话控制条（3种状态）
- 静音/保持/转接(4种)/咨询三方/DTMF/自助服务/话后处理/通话历史

**4.2.4 座席面板**：状态管理(4种) + 外部会话 + 通话视图
**4.2.5 引导教程(11步)**：欢迎 → 来电 → 连接 → 弹屏 → MCB → ACW → Widget → 面板 → 关闭 → 外呼 → 完成

### 五、文案
按模块分组的完整中英文对照表（约200+条翻译）
