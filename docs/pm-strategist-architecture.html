🚀 怎么用这个 Agent

1

最简用法 — 在 CodeBuddy 对话框中输入

/strategize 我们的电商平台需要一个智能推荐系统。 用户反馈"猜你喜欢"完全不准，点击率只有 2%，行业平均 8%。 团队 3 后端 + 2 前端，无 ML 工程师，Q3 必须上线，月预算 ≤ 20 万。 代码仓库：/path/to/ecommerce-platform
2

Agent 全自动执行五阶段流水线

无需你回答任何问题。它会自主读代码、查知识库、生成三版方案、批判打分、迭代修改、校验输出。

3

最终得到完整的 11 节产品策略文档

包含：问题定义 → 技术分析 → 三版方案对比 → 推荐 PRD → 路线图 → 风险清单 → 批判审查记录。

4

后续衔接（可选）

/generate-requirement # 细化推荐方案的 PRD /fullchain # 从 PRD 直接生成 UI 草稿 /strategy # 如需更广的市场战略分析

💡 重要说明：Skill ≠ 全部能力

项目自带 64 个 Skill（结构化产品方法论）+ ClawHub 社区 Skill（playwright-mcp、browser-use）。Agent 运行在 CodeBuddy IDE 环境中，IDE 还提供：
✅ web_search — 联网搜索（IDE 内置）
✅ web_fetch — 抓取网页内容（IDE 内置）
✅ read_file / codebase_search — 代码理解（IDE 内置）
✅ image_gen — 图片生成（IDE 内置）
✅ playwright-mcp / browser-use — 浏览器自动化（ClawHub 社区）

ClawHub 安装方式：
skillhub install playwright-mcp — Playwright MCP 服务器（截图+自动化）
skillhub install browser-use — 智能浏览器代理（AI 驱动交互）
